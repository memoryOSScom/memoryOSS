Test Results

These are real runner results generated from the same automated validation path used for releases.

The main public proof set is simple: retention under load, explicit long-memory regression, three reproducible everyday utility loops, sustained soak stability, compatibility across the supported Claude and Codex flows, and live quality metrics for retrieval and extraction.

Stable claims cover local everyday loops and guardrails, experimental claims cover retrieval and extraction tuning, and moonshot claims stay explicitly labeled instead of being smuggled into the stable path.

Public Benchmark Set

These are the benchmark classes that map most directly to user value and are the ones we intentionally surface publicly.

20k
Retention under load
Early high-signal memories remained retrievable across a 20k-memory corpus with strong signal retention and noise suppression.
REG
Long-memory regression
A sentinel stored first, followed by 5k or 20k background memories, remained retrievable at rank 1 in explicit regression runs.
SOAK
2-hour soak stability
Sustained workload completed without errors or leak signals while retrieval quality remained stable.
MATRIX
Compatibility matrix
Claude OAuth/API and Codex OAuth/API are tested across the supported MCP and proxy paths rather than left as undocumented assumptions.
LOOP
Everyday utility loops
Three reproducible loops now measure repeated-context elimination, portable project transfer, review throughput, replay fidelity, and blocked bad actions across HTTP API, CLI, reader, proxy, and two runtimes.
QUAL
Claim lanes
The live report below separates stable utility proofs from experimental retrieval/extraction deltas and moonshot product bets, so daily-value claims are backed by the rerunnable path rather than by aspirational demos.
TOKEN
Input-token efficiency
In a constrained 10-task Claude benchmark, recalled memory context reduced average input tokens by 44.4% versus replaying full task context. This is evidence for repeated-task context compression, not a universal token-savings guarantee.
